# 金融资产
标签:: #元 #经济  

定义:: 金融体系通过创造能够在金融市场上买卖的金融资产来实现功能。

  - 股权：股权是公司的所有权，通常形式为股票，拥有者享有股息。
	- 股票：一种代表一个公司的部分所有权的金融资产。
	- 股息：定期向公司全体股东进行的分红。
  - 债权：最基础和最常见的债权种类是贷款。
	- 贷款：是贷款人基于借款人还本付息的承诺同意借钱的协议。
	- 债券：代表发行者在约定到期日偿还票面价值并按约定利率付息的债务证明。债券是一种贷款，被标准化为更具交易性和流动性的资产。
  - 衍生品，是一种价值以其他资产的价值为基础的资产。衍生品意味着将风险转移给那些愿意承受的人。
  - 金融资产风险和收益：风险和收益通常是直接相关的。
	- 金融资产中存在两种类型的风险：
	  - 市场风险，整个市场广泛共享的风险；
	  - 个别风险，针对特定公司或特定资产的风险。
	- 资产组合可以有利于分数特定风险，某些市场风险存在于所有的资产组合中。

可大致分为4类:
1. 权益类资产。股票指数基金
2. 债权类资产。利率债指数基金
3. 大宗商品和REITs。以黄金、原油和REITs为代表。
4. 现金。以货币基金为代表
