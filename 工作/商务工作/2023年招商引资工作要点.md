# 2023 年招商引资工作要点
#记 #工作

> 2023 年是全面贯彻落实党的二十大精神开局之年，也是实施十四五规划承上启下之年。
总体要求:  
> 以习近平新时代中国特色社会主义思想为指导，全面贯彻落实党的二十大和中央经济会议精神，深入贯彻习近平总书记视察江西重要讲话精神，按照省委经济工作会议、市委市政府系列决策部署，聚焦“做示范、勇争先”目标要求，坚持全面贯彻落实新发展理念、全方位对外开放，树牢“项目为王”理念，立足于产业链延链补链强链壮链，大力开展目标化、清单化、精准化招商，引进一批规模大、质量优、效益好，符合我市发展战略的优强项目，为建设制造强市、打造区域中心，奋力开创现代化大美上饶新篇章提供强有力支撑。
目标任务：招商引资规模稳步增长、招商引资质量持续提升、重大项目加快落地。

## 重点工作
### 招商引资方式
1. 资本招商。发挥产业引导基金、创业投资基金作用，推进基金招商。
2. 专业招商。与专业招商机构和企业合作，与专业咨询公司合作。依托驻京办、驻沪办，开展驻点招商。借助已落户企业和饶商、商协会以商招商。
3. 主官招商。由一把手亲自开展招商活动，推动重大项目签约、落地。
### 招商引资行动
1. 产业链招商。围绕主导产业，开展产业链目标化、清单化精准招商。
2. 驻点招商，小分队招商。围绕长三角、粤港澳、京津冀，聚焦八条战线，开展驻点招商、小分队招商。
3. 依托资源禀赋，重视资源发掘和储备、项目整体布局和策划推介，吸引行业头部企业、央企共同开放。
4. 开展三清三回专项行动。
5. 走进跨国公司总部专项行动。
### 招商引资落地
1. 搭建招商活动平台。举办融入长三角、对接粤港澳大湾区、走进海西经济区、稳外资等专题招商活动。
2. 激发开发区承接潜力。创新开发区体制机制。加快建设跨境电商综试区、综合保税区等开放平台。
3. 推动招商项目落地。定期调度项目落户所需用地、用能、用气、环评、审批、金融等问题。
4. 优化营商环境。
