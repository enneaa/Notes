# 羽毛球大讲堂
#记 #运动  

**桃田贤斗技术特点**
- 控制底线 - 快连网前 - 伺机突击
	- 控制底线（基础）
		- 高球技术
		- 推挑技术
	- 快速连贯（做球）
		- 抢放
		- 抢勾
		- 重复推
	- 突击进攻（得分）
- 网前技术细致、前半场连贯快
- 突击技术一致性强、动作小、效率高
- 控球耐心、出球质量高、防守扎实  
**羽毛球中常见的攻击套路**
- 攻击套路：强攻、抢攻、突击
- 强攻
	- 来球到位
		- 推球、高球、挑球
		- 落点到位，底部边区
	- 底线组合强攻
		- 杀上网 推扑压
- 强攻易守：试探性进攻（软压）+ 快速组合进攻
- 抢攻：相持过程中率先发难发起组合进攻
	- 发球抢攻（第三拍）
	- 接发球抢攻（第四拍）
	- 中前场加速抢攻：快推、快勾、跳扑
	- 中后场加速抢攻：平高、快吊、扣杀
	- 前后组合式抢攻：平高、快吊 + 快推扑
- 突击：中后场抢攻
	1. 平高 + 扣杀、快拦吊
	2. 快推勾 + 扣杀、快拦吊
	3. 拦吊、软压 + 扣杀、快拦吊
	4. 组合进攻：强攻 + 突击
- 马林 世锦赛 三冠
	- 杀上网 - 推扑压打法
	- 平高球和推球控制底线，快勾搓放（做球）
	- 反克强攻
		- 底线反击能力强，限制上网
		- 灵活，防守好，适应平快球
		- 用弧度调控节奏，有突击
	- 戴资颖
		- 变线
		- 变速
		- 变技术
	- 底线控制能力
