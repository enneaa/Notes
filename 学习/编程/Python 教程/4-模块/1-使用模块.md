# 1- 使用模块

```python
#!/usr/bin/env python3
# -*- coding:utf-8 -*-
'a test module'
__author__='Shen Bo'
import sys
def test():
    args=sys.argv
    if len(args)==1:
        print('Hello, world!')
    else len(args)==2:
        print('Hello,%s!'%args[1])
    else:
        print('Too many argument!')
    if __name__=='__main__':
        test()
```

### 作用域
`__xxx__` 为特殊变量  
`_xx` ` __xx` 为非公开的 (private)
