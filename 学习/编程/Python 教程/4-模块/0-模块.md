# 0- 模块
### Module
使用模块的好处:
- 可维护
- 可复用
- 避免函数名及变量名冲突

  > [内置函数](http://docs.python.org/3/library/functions.html)

- 避免模块名重复

  > 按目录来组织模块的方法,称之为包 (Package)
  >
  > eg.
>
  > ```
  > mycompany
  > ├─ __init__.py
  > ├─ abc.py
  > └─ xyz.py
  > ```

创建模块的注意事项:
- 模块名遵循 Python 变量命名规范
- 模块名不与系统模块名冲突,可通过 `import abc` 判断系统是否存在该模块
