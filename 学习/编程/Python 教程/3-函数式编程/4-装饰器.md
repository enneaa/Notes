# 4- 装饰器

> 在代码运行期间动态增加功能的方式,称之为 " 装饰器 '(Decorator)

```python
def log(func):
    def wrapper(*args,**kw):
        print('call %s():',%func._name_)
        return func(*args,**kw)
    return  wrapper
@log
def now():
    print('2015-3-25')
```

```python
#将原始函数的_name_等属性复制到wrapper()函数中
import functools
def log(func):
    @funtools.wrap(func)
    def wrapper(*args,**kw):
        print('call %s():',%func._name_)
        return func(*args,**kw)
    return wrapper
```
