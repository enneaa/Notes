# 3- 递归函数

```python
def fact(n):
    if n==1:
        return 1
    return n*fact(n-1)
```

> 函数调用是通过栈（stack）这种数据结构实现的，每当进入一个函数调用，栈就会加一层栈帧，每当函数返回就会减一层栈帧。递归调用的次数过多时会导致栈溢出。
>
> 可通过 **尾递归** 优化，尾递归是指在函数返回的时候，调用自身本身，并且，return 语句不能包含表达式。这样编译器就可以把尾递归做优化，使递归本身无论调用多少次，都只占用一个栈帧，不会出现栈溢出的情况。

```python
def fact(n):
    return fact_iter(n,1)
def fact_iter(num,product):
    if num==1:
        return product
    return fact_iter(num-1,num*product)
```
