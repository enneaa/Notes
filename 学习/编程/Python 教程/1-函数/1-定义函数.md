### 参数检查

```python
def my_abs(x):
    if not isinstance(x,(int,float)):
        raise TypeError('bad openrand type')
    if x>=0:
        return x
    else:
        return -x
```

### 返回多个值

```python
import math #导入math包
def move(x,y,step,angle=0):
    nx=x+step*math.cos(angle)
    ny=y+step*math.sin(angle)
    return nx,ny
```

> Python 的函数返回多值就是返回一个 tuple
