# 99- 函数

| 函数         | 用法                                                      | 示例                                                 |
| ------------ | --------------------------------------------------------- | ---------------------------------------------------- |
| print()      | 打印                                                      | print('Hello,world')                                 |
| ord()        | 获取字符的整数表示                                        | ord('A')                                             |
| chr()        | 把编码转换为对应的字符                                    | chr(66)                                              |
| encode()     | 将 `str` 编码为指定的 `bytes`                                | 'ABC'.encode('ascii')                                |
| decode()     | 把 `bytes` 变为 `str`, 可传入 `errors='ignore'` 忽略错误的字节 | b'\xe4\xb8\xad\xff'.decode('utf-8', errors='ignore') |
| len()        | 计算 `str` 包含多少个字符                                   | len('ABC');len(b'ABC)                                |
| input()      | 读取用户的输入，返回 `str` 数据类型                         |                                                      |
| int()        | 将 `str` 转换为整数                                         |                                                      |
| range()      | 生成一个整数序列                                          |                                                      |
| list()       | 转换为 list                                               | list(range(5))                                       |
| isinstance() | 判断变量是否是某个类型                                    |                                                      |
| type()       | 判断对象类型                                              |                                                      |
