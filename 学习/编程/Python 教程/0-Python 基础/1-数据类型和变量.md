# 1- 数据类型和变量
### 数据类型和变量
数据类型：
1. 整数  
   十六进制 用 `0x` 前缀 `0-9`,`a-f` 表示，例 `0xa5b4c3d2`
2. 浮点数  
   科学记数法 `1.23e9`
3. 字符串  
   `'abc'`  
   `'I\'m \"OK\"'`  
   `\n` 表示换行，`\t` 表示制表符  
   `r''` 表示 `''` 内部的字符串默认不转义

   ```python
   #多行内容
   print('''line1
   ...line2
   ...line3''')
   ```

4. 布尔值  
   `True` `False`  
   布尔值可用 `and` `or` `not` 运算
5. 空值  
   `None`
6. 变量  
   `a=1`
7. 常量  
   `PI=3.14159265359`

> 任何数据都可看成一个对象，而变量就是在程序中指向这些数据对象的。
