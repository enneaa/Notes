# 3- 使用 List 和 Tuple
### List

> list 是一种有序的集合，可以随时添加和删除其中的元素

```python
classmates = ['Michael', 'Bob', 'Tracy']
classmates[0]
classmates[-1]
classmates.append('Adam')
classmates.insert(1, 'Jack')
classmates.pop()
classmates.pop(1)
classmates[1] = 'Sarah'
L = ['Apple', 123, True]
s = ['python', 'java', ['asp', 'php'], 'scheme']
```

### Tuple

```python
classmates = ('Michael', 'Bob', 'Tracy')
t = ()
t = (1,)
t = ('a', 'b', ['A', 'B'])
```

> list 和 tuple 是 Python 内置的有序集合，一个可变，一个不可变。
