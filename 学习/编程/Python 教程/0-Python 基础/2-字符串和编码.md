### 字符编码

> 8 个 bit 作为一个 byte，一个字节可表达的最大的整数就是 255（二进制 111111111=十进制 255）。用多个字节表示更大的整数。
>

- ASCII 编码
- Unicode 编码
- UTF-8 编码  

| 字符   | ASCII    | Unicode           | UTF-8                      |
| :--- | -------- | ----------------- | -------------------------- |
| A    | 01000001 | 00000000 01000001 | 01000001                   |
| 中    | x        | 01001110 00101101 | 11100100 10111000 10101101 |

> 计算机内存中，统一使用 Unicode 编码，但需要保存或需要传输的时候，就转换为 UTF-8 编码。

### Python 的字符串
Python 的字符串类型是 `str`。
Python 表示 `byte` 类型的数据：`x=b'ABC'`

> 当 Python 解释器读取源代码时，为按 UTF-8 编码读取需在文件开头写明：
>
> ```python
> #!/usr/bin/env python4
> # -*- coding: uft-8 -*-
> ```
>
> 还要确保文本编辑器正在使用 UTF-8 without BOM 编码

### 格式化
Python 中，用 `%` 实现格式化：

```python
'HI,%s,you have $%d.'%('Michael',1000000)
```

常见的占位符：

| 占位符  | 替换内容   |
| ---- | ------ |
| %d   | 整数     |
| %f   | 浮点数    |
| %s   | 字符串    |
| %x   | 十六进制整数 |

#### format()

```python
 'Hello, {0}, 成绩提升了 {1:.1f}%'.format('小明', 17.125)
```
