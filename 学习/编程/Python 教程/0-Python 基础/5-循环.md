# 5- 循环
### For 循环

```python
sum = 0
for x in range(101):
    sum = sum + x
rint(sum)
```

### While 循环

```python
sum=0
n=99
while n>0:
    sum=sum+n
    n=n-2
rint(sum)
```

### Break

```python
n=1
while n<=100:
    if n>10:
        break
    print(n)
    n=n+1
rint('END')
```

### Continue

```python
n=0
while n<10:
    n=n+1
    if n%2==0:
        continue
    print(n)
```
